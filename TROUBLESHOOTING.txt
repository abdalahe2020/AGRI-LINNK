🌿 اجري-تشين | دليل حل مشكلة تسجيل الدخول
═══════════════════════════════════════════════════════════════════

إذا كنت ترى رسالة "حدث خطأ في تسجيل الدخول. حاول مرة أخرى"، اتبع الخطوات التالية:

═══════════════════════════════════════════════════════════════════
الخطوة 1: تشغيل XAMPP
═══════════════════════════════════════════════════════════════════

1. افتح XAMPP Control Panel
2. شغّل Apache (Apache Server)
3. شغّل MySQL (MySQL Database)
4. تأكد من ظهور كلمة "Running" بجانبهم

═══════════════════════════════════════════════════════════════════
الخطوة 2: فحص النظام
═══════════════════════════════════════════════════════════════════

افتح هذا الرابط في المتصفح:
http://localhost/new/agri-chain/php/diagnostic.php

هذا سيخبرك بـ:
- إصدار PHP
- هل MySQL Extension مثبت
- هل قاعدة البيانات موجودة
- هل الجداول موجودة
- عدد المستخدمين الموجودين

إذا كان كل شيء أحمر (✗)، اتبع الحل المقترح.

═══════════════════════════════════════════════════════════════════
الخطوة 3: إنشاء قاعدة البيانات
═══════════════════════════════════════════════════════════════════

افتح phpMyAdmin في متصفحك:
http://localhost/phpmyadmin

ثم:
1. افتح "التشغيل السريع" (Execution) أو "الاستعلام" (Query)
2. الصق محتوى ملف:
   g:\xampp\New folder\htdocs\new\agri-chain\php\database.sql
3. اضغط Execute

أو الطريقة الأخرى:
1. اضغط "قاعدة بيانات جديدة" (New Database)
2. اكتب اسم: agrichain
3. اختر Character Set: utf8mb4
4. اضغط Create
5. ثم افتح تبويب Import واختر ملف database.sql

═══════════════════════════════════════════════════════════════════
الخطوة 4: إنشاء مستخدمين اختباريين
═══════════════════════════════════════════════════════════════════

افتح هذا الرابط في المتصفح:
http://localhost/new/agri-chain/php/create_test_user.php

سيتم إنشاء:
- مزارع: test@example.com / password123
- تاجر: trader@example.com / trader123
- مدير: admin@example.com / admin123

═══════════════════════════════════════════════════════════════════
الخطوة 5: اختبار تسجيل الدخول
═══════════════════════════════════════════════════════════════════

جرّب الاختبار القائم بذاته:
http://localhost/new/agri-chain/php/test_login.php

هذا الاختبار يستخدم نفس طريقة الواجهة الأمامية لكن بدون HTML/CSS
سيخبرك بالضبط ما هي المشكلة

أو جرب الواجهة الأمامية:
http://localhost/new/agri-chain/login.html

استخدم البيانات الاختبارية:
البريد: test@example.com
كلمة المرور: password123

═══════════════════════════════════════════════════════════════════
استكشاف الأخطاء الإضافية
═══════════════════════════════════════════════════════════════════

إذا استمرت المشكلة:

1. افتح جهاز التطوير (F12 في المتصفح)
2. اذهب إلى تبويب "Console"
3. حاول تسجيل الدخول مرة أخرى
4. انسخ الخطأ الذي يظهر

أو:

1. افتح ملف السجلات:
   g:\xampp\apache\logs\error.log
   g:\xampp\mysql\data\error.log
2. ابحث عن الأخطاء الأخيرة
3. همّ قد تخبرك بما هي المشكلة

═══════════════════════════════════════════════════════════════════
الملفات الهامة
═══════════════════════════════════════════════════════════════════

php/diagnostic.php ........... فحص النظام الشامل
php/create_test_user.php ..... إنشاء مستخدمين اختباريين
php/test_login.php ........... اختبار تسجيل الدخول
php/database.sql ............. ملف إنشاء قاعدة البيانات
php/login.php ................ ملف معالجة تسجيل الدخول
js/auth.js ................... ملف JavaScript لنموذج التسجيل

═══════════════════════════════════════════════════════════════════
الأسباب الشائعة للخطأ
═══════════════════════════════════════════════════════════════════

❌ XAMPP لم يتم تشغيله
   → شغّل Apache و MySQL من XAMPP Control Panel

❌ قاعدة البيانات غير موجودة
   → افتح phpMyAdmin وأنشئ DATABASE: agrichain

❌ الجداول غير موجودة
   → استورد ملف database.sql في phpMyAdmin

❌ لا يوجد مستخدمين
   → افتح http://localhost/new/agri-chain/php/create_test_user.php

❌ رقم الهاتف غير صحيح
   → استخدم رقم هاتف مصري بصيغة: 01XXXXXXXXX

❌ كلمة المرور قصيرة جداً
   → استخدم 8 أحرف على الأقل

═══════════════════════════════════════════════════════════════════
الدعم
═══════════════════════════════════════════════════════════════════

إذا استمرت المشكلة:
1. جرّب اختبار النظام: diagnostic.php
2. جرّب اختبار تسجيل الدخول: test_login.php
3. تحقق من سجلات الأخطاء
4. تأكد من تشغيل XAMPP
